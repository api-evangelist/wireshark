We are proud to announce the release of Wireshark 1.12.0! This is a fairly significant release for us, as we expect it to be the very last release using the GTK toolkit in the default interface (see this post for our plans to replace it). Wireshark 1.12 contains a substantial number of fixes and new features, representing just over a year of development effort.
