Few days ago, I learned about a cool visualization program called Codeswarm which, surprisingly, is made by a guy that lives in Davis California like me. Codeswarm can be fed with the logs from a source code repository and produces an animated history of that source code. Soon enough, my weekend project became the creation a video that would condense the 11+ years of the Wiresahrk source code in 3 minutes.
